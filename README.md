## Run Locally

Clone the project

```bash
  git clone https://github.com/albandelishi/chat-nodejs-socketio.git
```

Go to the project directory

```bash
  cd chat-nodejs-socketio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node server.js or nodemon
```
